# Keyboard Shortcuts

Master these keyboard shortcuts to work faster in Nativeline and the iOS Simulator.

## Nativeline Shortcuts

### Chat Interface

| Shortcut | Action |
|----------|--------|
| **Enter** | Send message |
| **Shift+Enter** | New line in message |
| **Cmd+Enter** | Quick send (alternative) |

### Navigation

| Shortcut | Action |
|----------|--------|
| **Cmd+1** | Go to Chat tab |
| **Cmd+2** | Go to Settings tab |
| **Cmd+3** | Go to Database tab |
| **Cmd+4** | Go to Deploy tab |

### General

| Shortcut | Action |
|----------|--------|
| **Cmd+,** | Open Preferences |
| **Cmd+N** | New Project |
| **Cmd+O** | Open Project |
| **Cmd+W** | Close Window |
| **Cmd+Q** | Quit Nativeline |

## iOS Simulator Shortcuts

### Device Control

| Shortcut | Action |
|----------|--------|
| **Cmd+Shift+H** | Home button (once) |
| **Cmd+Shift+H** (2x) | App switcher |
| **Cmd+L** | Lock screen |
| **Cmd+Shift+L** | Unlock screen |

### Rotation

| Shortcut | Action |
|----------|--------|
| **Cmd+Left Arrow** | Rotate left |
| **Cmd+Right Arrow** | Rotate right |

### Appearance

| Shortcut | Action |
|----------|--------|
| **Cmd+Shift+A** | Toggle light/dark mode |

### Screenshots & Recording

| Shortcut | Action |
|----------|--------|
| **Cmd+S** | Take screenshot |
| **Cmd+R** | Start/stop screen recording |

### Keyboard

| Shortcut | Action |
|----------|--------|
| **Cmd+K** | Toggle software keyboard |

### Window Scaling

| Shortcut | Action |
|----------|--------|
| **Cmd+1** | Actual Size (100%) |
| **Cmd+2** | Fit to Window |
| **Cmd+3** | 50% Scale |

### Gestures

| Shortcut | Action |
|----------|--------|
| **Option+Click and Drag** | Pinch to zoom |
| **Option+Shift+Click and Drag** | Two-finger drag/pan |

## Tips for Efficient Shortcuts

### Learning New Shortcuts

1. Focus on one category at a time
2. Practice the most common ones first:
   - **Enter** to send messages
   - **Cmd+Shift+H** for Simulator home
   - **Cmd+S** to save
3. Add more as they become natural

### Creating Custom Shortcuts

macOS lets you create custom shortcuts:

1. Open **System Settings → Keyboard → Keyboard Shortcuts**
2. Select **App Shortcuts**
3. Click **+**
4. Choose Nativeline and add your shortcut
